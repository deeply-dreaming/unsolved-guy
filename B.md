B1: Perfect numbers: Do any odd perfect numbers exist (and are there a-k-perfect numbers with k >= 4 / infinitely many)?
B2: Almost perfect / quasi-perfect / weird / harmonic numbers: Are there almost perfect numbers other than powers of 2, or any quasi-perfect numbers?
B3: Unitary perfect numbers: Are there infinitely many unitary perfect numbers?
B4: Amicable numbers: Are there any odd amicable numbers not divisible by 3 (and how small can an odd amicable pair be)?
B5: Betrothed (quasi-amicable) numbers: Do there exist betrothed numbers (m, n) of the same parity?
B6: Aliquot sequences: Do some aliquot sequences (iterating s(n) = σ(n) - n) grow without bound, or are all sequences bounded/terminating?
B7: Aliquot cycles / sociable numbers: Do any aliquot 3-cycles exist, and are there infinitely many k-cycles for each k?
B8: Unitary aliquot sequences: Are there unbounded unitary aliquot sequences (and are related iterates like w(n) bounded)?
B9: Superperfect numbers: Are there any odd superperfect numbers (and are there quasi-/almost-superperfect numbers beyond known families)?
B10: Untouchable numbers: Is 5 the only odd untouchable number, and can there be arbitrarily long runs of consecutive even untouchables?
B11: Solutions of mσ(m) = nσ(n): Are there infinitely many primitive solutions (not from scaling), and is m/n bounded for solutions?
B12: Analogs with d(n) and σ_k(n): Do there exist distinct m, n such that mσ_k(m) = nσ_k(n) (e.g., for k = 2)?
B13: Solutions of σ(n) = σ(n + 1): Does σ(n) = σ(n + 1) (or σ(n) = σ(n + ℓ)) hold infinitely often for some shift ℓ?
B14: Irrational series: Is ∑_{n≥1} σ_k(n)/n! irrational (beyond the known cases k = 1 and 2)?
B15: Solutions of σ(q) + σ(r) = σ(q + r): Are there infinitely many (primitive) solutions, and how many are there with q + r < x?
B16: k-full (powerful) numbers: Are there infinitely many pairs of consecutive k-full numbers, and finitely many powerful n with n^2-1 also powerful?
B17: Exponential-perfect numbers: Is there an e-perfect number not divisible by 3 (equivalently 36), and do any e-multiperfect numbers exist?
B18: Solutions of d(n) = d(n + 1): How long can a run of consecutive integers with the same number of divisors be?
B19: Prime-factor coincidences / abc: Can (m, n+1) and (m+1, n) share the same prime factors nontrivially, and is rad(ABC)^{1+ε} < C only finitely often for coprime A+B=C?
B20: Cullen and Woodall numbers: Can p and p·2^p + 1 both be prime (and related questions for n·2^n ± 1)?
B21: Sierpinski problem: What is the least k such that k·2^n + 1 is composite for all n?
B22: Factorial n as a product of n large factors: In writing n! as a product of n integers, how large can the smallest factor be (and how big can gaps/plateaus be)?
B23: Equal products of factorials: Are there any nontrivial solutions to n! = a1! a2! ··· ar! beyond the known small examples?
B24: Largest subset with no member dividing two others: How large can f(n) be for subsets of [1, n] in which no element divides two other elements?
B25: Equal sums of geometric progressions with prime ratios: Is 31 the only prime representable in two ways as (p^r - 1)/(p - 1) with p prime and r ≥ 3?
B26: Coprime-free sets: What is the maximum size of a subset of [1, n] containing no ℓ pairwise coprime integers (and is Erdos’s conjectured construction optimal)?
B27: New prime factors in short blocks: Does v0(n) (the maximal count of prime factors of n + k not dividing any n + i for 0 ≤ i < k) tend to infinity with n?
B28: Consecutive numbers with distinct prime factors: Do there exist n consecutive integers each having either two distinct prime factors < n or a repeated prime factor < n?
B29: Prime divisors of x + 1, …, x + k: Is there a fixed k (perhaps k = 3) that uniquely determines x from the sets of prime divisors of x + 1, …, x + k?
B30: Square products in short intervals: How large must t_n be so that among n + 1, …, n + t_n there is a subset whose product (times n) is a square?
B31: Binomial coefficients: Are there only finitely many binomial coefficients with deficiency 1 (and how does the Erdos-Selfridge function g(k) grow)?
B32: Grimm’s conjecture: In any run of consecutive composite integers n + 1, …, n + k, can one choose distinct primes p_j with p_j | (n + j) for each j?
B33: Divisors of a binomial coefficient: How large can the largest divisor < n of C(n, k) be, and is the central binomial coefficient ever squarefree for n > 4?
B34: Divisibility in binomial coefficients: For which k is it true that for all n ≥ 2k there exists i < k with (n - i) | C(n, k)?
B35: Consecutive blocks with the same prime factors: Can (m+1)…(m+k) and (n+1)…(n+ℓ) have exactly the same set of prime divisors infinitely often?
B36: Euler’s totient function: Are there infinitely many n such that φ(n) = φ(n + 1)?
B37: Lehmer’s totient divisibility problem: Can φ(n) properly divide n - 1 for any composite n?
B38: Solutions of φ(m) = σ(n): Are there infinitely many pairs (m, n) such that φ(m) = σ(n)?
B39: Carmichael’s conjecture: Does any totient value occur exactly once (i.e., is φ(n) = m never uniquely solvable)?
B40: Jacobsthal gaps: How large can J(n), the maximum gap between integers coprime to n, be?
B41: Iterations of φ and σ (and related averages): Can iterating (σ(n) + φ(n))/2 produce sequences that grow indefinitely without fracturing into nonintegers?
B42: Compositions of φ and σ: Are there infinitely many solutions to φ(σ(n)) = n (equivalently σ(φ(m)) = m), and how does σ(φ(n))/n behave?
B43: Alternating sums of factorials: Are there infinitely many primes of the form n! - (n - 1)! + (n - 2)! - ··· ± 1! (and related divisibility questions)?
B44: Sums of factorials (Kurepa’s conjecture): Is !n = 0! + 1! + ··· + (n - 1)! never divisible by n for all n > 2?
B45: Euler numbers: For primes p ≡ 1 (mod 8) (or p ≡ 5 (mod 8)), is E_{(p-1)/2} ≠ 0 (mod p)?
B46: Largest prime factor P(n): Are there infinitely many primes p such that (p - 1)/P(p - 1) is a power of 2 (or 2^k·3)?
B47: Exponent divisibility: For which a > b does 2^a - 2^b divide n^a - n^b for all integers n?
B48: Primorial prime-product integrality: For which m is ∏_{n=1}^m p_n/(p_n-1) an integer (known m = 1,2,3,4,8)?
B49: Smith numbers: Are there infinitely many composite Smith numbers (and more generally, k-Smith numbers) in a given base?
B50: Ruth-Aaron numbers: How common are consecutive integers n, n + 1 with equal sums of prime factors (counting multiplicity)?
